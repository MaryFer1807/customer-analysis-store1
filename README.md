## Project Overview
In this second stage of the Store 1 project, the focus shifts from data preparation to in-depth customer analysis. Building on the cleaned dataset from Sprint 1, this project analyzes customer behavior to support more personalized and effective marketing campaigns.

The goal is to better understand purchasing patterns, customer segments, and consumption trends over time.

## Objective
Analyze customer data to:
- Identify customer segments based on behavior and spending patterns.
- Explore consumption trends across different product categories.
- Generate insights to support targeted advertising strategies.
- Strengthen business decision-making through data analysis.

## Analysis Performed
- Customer segmentation based on age, spending, and purchase categories.
- Analysis of total and category-level spending behavior.
- Identification of trends and patterns in customer consumption.
- Preparation of insights to support marketing personalization.

## Dataset
This analysis uses the cleaned and structured dataset produced in **Sprint 1**, ensuring data consistency and reliability throughout the analysis process.

## Tools Used
- Python
- Jupyter Notebook
- Data analysis techniques
- Exploratory analysis and segmentation logic

## Business Value
The insights generated in this project help Store 1:
- Personalize marketing campaigns.
- Improve customer targeting.
- Better understand customer behavior and preferences.
- Support strategic decision-making.

This project represents the analytical phase of a multi-stage data project, with advanced data manipulation and analysis continuing in the following sprint.
